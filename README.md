# AdaSubGram

[![Build Status](https://github.com/amousavigourabi/AdaSubGram.jl/actions/workflows/CI.yml/badge.svg?branch=master)](https://github.com/amousavigourabi/AdaSubGram.jl/actions/workflows/CI.yml?query=branch%3Amaster)

Julia package to build subword-aware adaptive skipgram embeddings. This work is a combination of the ideas of [Bartunov et al. (2016)](https://proceedings.mlr.press/v51/bartunov16.html) and [Bojanowski et al. (2017)](https://doi.org/10.1162/tacl_a_00051).

This package is not in the general repository, so it must be installed with `Pkg` through the URL `https://github.com/amousavigourabi/AdaSubGram.jl.git`.
